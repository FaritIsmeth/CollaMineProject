What is CollaMine Project about?

It is created to address two important issues found in the domain of big data analysis:

Content collection

Content extraction

This project will serve as a common platform for archiving and extracting online documents and Social Network System content.

We achieve this by implementing a smart cache mechanism which shares the archived contents to all clients.

This not only saves consumers from spending extra resource in online content downloading, but also eliminates redundant copies of documents via duplicate detection.
